# Quora clone

Quora clone has two file Frontend and Backend.

## Installation

You have to install all dependencies for each separately 
```bash
cd backend
```
```bash
cd frontend
```

```bash
npm install
```

## Usage

For frontend
```Start
npm run dev
```
For backend
```Start
nodemon server.js
```
